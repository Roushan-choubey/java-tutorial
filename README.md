# Java Tutorial Practice Projects

This repository contains beginner-friendly Java programs organized by topic and chapter.
It includes examples for:

- Basics and classes
- Constructors, inheritance, and method overriding
- Collections: ArrayList, HashSet, TreeSet, Stack, Queue, Deque, Map, and Vector
- Exception handling
- Multithreading

## Structure

- 1/ - Basic Java programs
- 2/ - Collections and data structures
- 3/ - Input handling and maps
- 4/ - Autoboxing and basic exceptions
- 5/ - Advanced exception handling examples
- 6/ - PrintWriter examples
- 7/ - Thread example

## How to run

Compile and run Java files with:

```bash
javac FileName.java
java FileName
```

Replace FileName with the target class name.
