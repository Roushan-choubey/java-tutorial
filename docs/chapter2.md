# Chapter 2: Collections and data structures

These programs demonstrate common collection classes in Java:

- ArrayList for dynamic lists
- HashSet for unique values
- TreeSet for sorted values
- Stack for LIFO operations
- Queue and Deque for ordered processing
- Iterator usage for traversal

- 2/array/setinterface.java
